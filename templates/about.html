About Flagship

Flagship is a fun and educational game designed to help players learn about the flags of countries around the world. The game challenges players to correctly identify country flags from multiple choice options within a 90-second time limit. It provides an engaging way to learn geography and improve your knowledge of world flags.

I created this game out of a personal interest and admiration for flags and the nations they represent. Flags are powerful symbols of a countryâ€™s heritage, culture, and pride. Through this game, I aim to share my fascination with flags and promote a greater understanding of the diversity in our world.

The game is designed to help players learn about different flags and appreciate the uniqueness of each nation. By playing, I hope you will not only enjoy the challenge but also gain knowledge about the flags and the countries they symbolize.

Using the Wikipedia API

The Wikipedia API is used to provide links to more information about each country. When you click on the name of a country in the Learn Flags section, it takes you to the respective Wikipedia page for that country.

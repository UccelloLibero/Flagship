About Flagship

Flagship is a fun and educational game designed to help players learn about the flags of countries around the world. The game challenges players to correctly identify country flags from multiple choice options within a 60-second time limit. It provides an engaging way to learn geography and improve your knowledge of world flags.

I created this game out of a personal interest and admiration for flags and the nations they represent. Flags are powerful symbols of a countryâ€™s heritage, culture, and pride. Through this game, I aim to share my fascination with flags and promote a greater understanding of the diversity in our world.

The game is designed to help players learn about different flags and appreciate the uniqueness of each nation. By playing, I hope you will not only enjoy the challenge but also gain knowledge about the flags and the countries they symbolize.

Python and Flask Methods

The backend of this game is built using Python and Flask. Here are some of the key methods used:

  • generate_options(correct_name): Generates a list of three options, including the correct flag name and two random incorrect ones.
  • start_new_game(): Initializes a new game session by selecting a random country and generating options for it.
  • @app.route('/check_answer', methods=['POST']): Endpoint to check the player's answer and update the correct and incorrect guess counters.
  • @app.route('/start_game', methods=['POST']): Endpoint to start a new game session and send the flag image and options to the client.
